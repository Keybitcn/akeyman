博客后台配置说明

后台使用 Decap CMS，需要一次性配置 GitHub 登录授权。

1. 创建 GitHub OAuth 应用

  1. 打开 GitHub OAuth 应用创建页
  2. Application name：随意，例如 Blog Admin
  3. Homepage URL：https://quna.fun
  4. Authorization callback URL：https://quna.fun/callback
  5. 创建后记下 Client ID 和 Client Secret

2. 在 Cloudflare Pages 添加环境变量

进入 Cloudflare 控制台 → Pages → 你的项目 → Settings → Environment variables，添加：

GITHUB_CLIENT_ID = 你的 Client ID
GITHUB_CLIENT_SECRET = 你的 Client Secret
ALLOWED_ORIGIN = quna.fun

保存后重新部署一次。

3. 打开后台

返回后台，点击 Login with GitHub 登录即可写文章。

本地试用

npm run dev
npm run cms

然后访问 http://localhost:3000/admin/
